Web and Mobile documentation
==================================

This is documentation for Web and Mobile mobile add-on product for `Plone CMS <http://plone.org>`_.

.. warn ::

   The active development on this project has ceased. The project has been migrated
   to Github. You are welcome to submit patches through Github and open an issue
   against a package to ask a new release.

The future Plone mobile development continues in `Plomobile project <https://github.com/miohtama/plomobile>`_.
