Go Mobile for Plone multichannel CMS features
-----------------------------------------------------

Introduction
-----------

How Go mobile for Plone multichannel CMS will solve
your mobile publishing and content management needs.
        
* Go Mobile for Plone multichannel CMS is based on award-winning, open source, 
  :doc:`Plone content management system </usermanual/plone>` technology
  
* :doc:`Open source </usermanual/license>` licensing guarantees that you stay free
  from vendor lock in the future
  
* :doc:`You can have single URL address for web and mobile site </usermanual/redirector>`_ - the user will be
  catered correctly no matter on which device or browser is being used
  
* High quality :doc:`extensible </developer/plugins>` code base with test
  suite over hundred tests and coverage 

* Publish to both web and mobile through easy user interface 

* Provide :doc:`optimal mobile browsing experience </usermanual/optimal_experience>`_
  for your site visitors.
      
* `Multichannel and convergence </usermanual/convergence>` features allow you to 
  make different versions of the same content for web and mobile
    
* Supports thousands of :doc:`mobile phones models </usermanual/sniffing>`_ from
  low end handsets to super smartphones. 
  
* :doc:`Mobile browser preview </usermanual/preview>`
  allows you to see how the page will look on the mobile
  when you are still writing it

* :doc:`Theming </usermanual/themes> compatible for different handsets
  
* Leverage existing hundreds of `Plone add-on products <http://plone.org/products>`_ for your own needs:
  blogs, forums, polls, intranets, enterprise network authentication...
